# stayora

## Description

A web platform for Stayora, an eco-tourism service focused on sustainable, nature-inspired accommodations.

## Tech Stack

- **Language:** JavaScript
- **Framework:** Express
- **Database:** MongoDB
- **Authentication:** Passport (utilizing Passport, `passport-local`, and `passport-local-mongoose`)
- **Storage:** Cloudinary (integrated with `multer` and `multer-storage-cloudinary`)
- **Package Manager:** npm
- **Key Dependencies:**
  - `@mapbox/mapbox-sdk`
  - `cloudinary`
  - `connect-flash`
  - `connect-mongo`
  - `dotenv`
  - `ejs`
  - `ejs-mate`
  - `express`
  - `express-session`
  - `joi`
  - `method-override`
  - `mongoose`
  - `multer`
  - `multer-storage-cloudinary`
  - `passport`
  - `passport-local`
  - `passport-local-mongoose`

## Installation

To install project dependencies:

```bash
npm install
```
